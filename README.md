# Loja Versículos ✝️

Site institucional e catálogo online para a **Loja Versículos**, uma marca de camisas católicas com estampas devocionais, carrinho de compras e finalização de pedido pelo WhatsApp.

## 📌 Sobre o projeto

A Loja Versículos é um projeto desenvolvido com foco em apresentar coleções de camisas católicas de forma elegante, responsiva e funcional.

O site permite que o usuário visualize produtos, escolha tamanhos, adicione itens ao carrinho e finalize o pedido diretamente pelo WhatsApp.

## 🚀 Funcionalidades

- Página inicial institucional
- Seção sobre a marca
- Catálogo de produtos
- Produtos carregados via `produtos.json`
- Separação por categorias
- Carrinho de compras
- Seleção de tamanhos
- Cálculo automático do total
- Finalização do pedido via WhatsApp
- Rodapé com contatos e redes sociais
- Layout responsivo para desktop, tablet e celular

## 🛒 Como funciona o pedido

- O cliente navega pelo catálogo.
- Escolhe a camisa desejada.
- Seleciona o tamanho.
- Adiciona o produto ao carrinho.
- Revisa o pedido.
- Finaliza pelo WhatsApp.

## 📦 Categorias de produtos

O catálogo pode ser organizado por categorias como:

- Devocional
- Mariana
- Bem-aventuranças
- Santos
- Camisas personalizadas

## 📱 Responsividade

O projeto foi pensado para funcionar em diferentes tamanhos de tela, com foco especial em dispositivos móveis.

## 🔗 Finalização via WhatsApp

O carrinho gera uma mensagem automática com:

- Nome do cliente
- E-mail
- Produtos escolhidos
- Tamanhos
- Quantidades
- Valor total

##✨ Identidade visual

A estética do site utiliza uma composição elegante, com cores sóbrias e detalhes dourados, reforçando a proposta de uma moda católica minimalista e contemporânea.
